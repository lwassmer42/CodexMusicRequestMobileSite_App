# CodexMusicRequestMobileSite_App (Music Request Tracker)

Mobile-first music request tracker with local storage and Excel/JSON import/export.



## GitHub Pages

This project is configured to deploy as a static site (SPA) to GitHub Pages:

- Uses hash routing (`#/home`) so refresh/deep links work on Pages.
- Uses a relative production base (`./`) so assets load correctly from `https://<user>.github.io/<repo>/`.

Workflow: `.github/workflows/deploy-pages.yml`
